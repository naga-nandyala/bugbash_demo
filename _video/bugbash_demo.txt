Imagine a bug bash that runs itself—every result, logged automatically. That’s the power of the AI-guided Bug Bash, seamlessly integrated in vee yes Code and driven by GitHub Copilot.

Here's how it works. You open Copilot Chat. It reads your test steps. Then it walks you through each one.

Four modes control the flow. Auto runs immediately. Interactive runs and asks for confirmation. Manual shows the command for you to run yourself. Destructive waits for your approval before executing.

Every step is logged to a timestamped markdown file. At the end, a summary shows what passed, failed, and skipped.

You observe. Copilot orchestrates. Everything is captured. Nothing is missed.
